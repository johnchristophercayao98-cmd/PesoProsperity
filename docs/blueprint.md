# **App Name**: PesoProsperity

## Core Features:

- Financial Goal Setting: Set specific financial objectives and visually track progress. Allows linking of savings and debt reduction efforts.
- Automated Budget Creation with AI tool: AI tool generates initial monthly budgets based on uploaded financial data and suggests adjustments. The AI tool decides whether to use the imported data.
- Recurring Transaction Management: Set up and manage recurring income and expense transactions with customizable frequencies and approval options.
- Budget vs. Actual Variance Analysis: Compare budgeted amounts against actual expenses and display variance results with alerts for overspending.
- Cash Flow Forecasting: Generate easy-to-understand cash flow forecasts with reminders to maintain a cash reserve.
- Debt Tracker: Monitor debt payoff progress with an interface using Philippine Peso sign.
- Comprehensive Financial Reporting: Generate detailed financial reports for specified periods, including income vs. expense summaries, budget variance reports, debt payoff progress, and cash flow forecasts.
- Spreadsheet Import/Export: Support importing and exporting data using spreadsheet files.
